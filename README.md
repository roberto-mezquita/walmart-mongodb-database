# Walmart MongoDB Database

Proyecto académico enfocado en el diseño e implementación de una base de datos NoSQL utilizando MongoDB, tomando como referencia la estructura operativa de una cadena comercial similar a Walmart.

## Descripción

El proyecto tuvo como objetivo modelar diferentes procesos comerciales mediante colecciones de documentos, aplicando conceptos de bases de datos NoSQL, operaciones CRUD, índices y agregaciones para el análisis de información.

La solución fue desarrollada utilizando MongoDB y JavaScript, permitiendo automatizar la inserción de datos y ejecutar consultas avanzadas sobre grandes volúmenes de información.
